# SOS Project

Right now, this project displays an RSS feed with musical accompaniment, using syllabes per word for rhythmic input.

### Dependencies

- ChucK (http://chuck.cs.princeton.edu/)
- Python
- Python packages (via pip):
  - Pyphen
  - pyOSC
  - beautifulsoup4
  - feedparser

### Running

    chuck demo.ck &
	python demo.py
